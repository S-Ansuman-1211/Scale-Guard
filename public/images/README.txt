Drop your image files in this folder. Expected filenames (referenced in src/data/media.js):

  cooling-towers.jpg   — Home hero (the dusk cooling-tower photo)
  valve.jpg            — Home case study + How It Works (the stainless inline device)
  building.jpg         — About page (IIT / head office building)
  nano-diagram.png     — optional science diagram

Any size works (images are object-fit: cover). Recommended ~1200px wide, JPG/PNG/WebP.
If a file is missing, the slot shows a navy gradient fallback instead of breaking.
To use different names, edit the paths in src/data/media.js.
